## Build Setup

```bash
# install node (recommend 16.16.0)
visit web https://nodejs.org/

# install expo-cli
$ npm i -g expo-cli (with administrator)

# install eas-cli for build
$ npm install -g eas-cli (with administrator)
```

## Development

```bash
Step 1: make sure you have package
- yarn
Step 2: run simulator or open device IOS/ANDROID have Expo
Step 3: run source
- expo start
# or
- yarn run start
```
